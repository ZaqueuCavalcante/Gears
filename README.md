## 13.2 - Nomenclature
- Para engrenagens cilíndricas de dentes retos:
    - Atributos fundamentais:
        - Número de dentes (N);
        - Diâmetro primitivo (d) [in ou mm];
    - Atributos derivados:
        - Módulo (m = d/N) [mm];
        - Passo diametral (P = N/d);
        - Passo circular (p = PI*d/N);
